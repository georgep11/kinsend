## Quick start
```bash
  npm install
  npm run start
  npm run build
```

## subdomain on local
If you are using Macbook Device
```bash
  sudo nano /etc/hosts
```
Add Host
[cnametext] depend your CNAME. This is an example only.
```bash
  127.0.0.1 kinsend.co cnametext.kinsend.co
```

Please run `kinsend.co:3000` or `cnametext.kinsend.co:3000` for testing
